Ingeniería del SAU

En el desarrollo del **Sistema Administrativo Unificado**, el objetivo fue centralizar la lógica de negocio académica. Se implementó una arquitectura MVC robusta para manejar miles de registros estudiantiles.

Se diseñó un esquema de base de datos relacional optimizado para evitar la redundancia y garantizar la integridad referencial en los reportes financieros.

Logro: Reducción del 40% en tiempos de procesamiento de matrículas.

Módulo de Inteligencia

Este proyecto se centró en la capa de presentación de datos. Se desarrollaron conectores de datos para transformar tablas SQL planas en objetos JSON dinámicos consumibles por el frontend.

Se aplicaron principios de UX para que los directivos pudieran identificar patrones de deserción mediante filtros interactivos y mapas de calor.
