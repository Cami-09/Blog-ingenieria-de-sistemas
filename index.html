FULL STACK DEVELOPMENT

Sistema Administrativo Unificado (SAU)

El Desafío: Las instituciones educativas manejaban procesos fragmentados, causando pérdida de datos y lentitud en los reportes gubernamentales.

La Solución: Desarrollé un núcleo centralizado con Python y SQL Server que unifica la gestión académica, financiera y administrativa en un solo flujo de trabajo.

  • Sincronización en tiempo real de notas y pagos.
  • Módulo de seguridad con roles jerárquicos (RBAC).
  • Exportación automática de informes en formatos legales.

DATA ENGINEERING & BI

Dashboard de Analítica Avanzada

El Desafío: Los directivos no podían visualizar tendencias de deserción o rendimiento estudiantil debido a que los datos estaban "crudos".

La Solución: Implementé un pipeline de datos que extrae información del SAU, la procesa y la visualiza mediante gráficos dinámicos interactivos.

  • Visualización de KPIs mediante Chart.js y D3.js.
  • Predicción de riesgos académicos mediante lógica estadística.
  • Interfaz responsive para acceso desde móviles.
