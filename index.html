Backend & SQL Server

Implementación del Ecosistema SAU

La creación del **Sistema Administrativo Unificado** nació de la necesidad de centralizar la información. Aquí se diseñó un motor de base de datos capaz de soportar registros de miles de estudiantes con integridad total.

Se implementaron vistas de usuario personalizadas que permiten a los administrativos generar reportes complejos en un solo clic, eliminando el error humano en un 90%.

Data Visualización

Dashboards de Inteligencia

Para este proyecto, el enfoque fue el **Business Intelligence**. Se integraron librerías de visualización dinámicas que extraen datos del núcleo del software para presentarlos de forma estratégica.

Cada gráfico fue diseñado pensando en la toma de decisiones gerenciales, permitiendo ver tendencias de crecimiento y operatividad en tiempo real.
